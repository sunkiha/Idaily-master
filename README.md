DAILY
==============================================
DAILY是一个非官方的基于MVVM架构的知乎日报APP。所有内容均来自网络，仅供学习使用。

[![Get it on Google Play](art/download.png)][1]

##项目结构
 * android data binding来实现MVVM。
 * dagger2来完成依赖注入。
 * retrofit2+rxjava实现restful的http请求。

##第三方类库
* retrofit2
* dagger2
* butterknife
* rxjava+rxandroid
* LoganSquare
* glide
* android data binding
* android support design
